# Reference dashboards

Rogue Dashboard is an original implementation. No source code or bundled assets from the projects below are included.

| Project | Workflows considered |
| --- | --- |
| Dashy | Search, theming, status and flexible layouts |
| Fenrus | Personal home pages and local configuration |
| Flame | Simple built-in application and bookmark editing |
| Heimdall | Application tiles and recognisable service presentation |
| Homarr | Drag-and-drop boards and visual configuration |
| Homepage | Service grouping, Docker awareness, widgets and YAML migration |
| Homer | Portable configuration and lightweight deployment |
| Organizr | Navigation, user access and reverse-proxy workflows |

Upstream licences and project behaviour can change. Repeat the licence review before any future third-party source or asset is proposed for inclusion.
